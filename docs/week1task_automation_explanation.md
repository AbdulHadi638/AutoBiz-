# Week 1: How Automation Replaces Manual Work

Automation takes repetitive tasks that humans normally do—like entering customer information, sending confirmation emails, and notifying the team—and does them automatically. 

This means no one has to spend time copying data or remembering follow-ups. As a result, work gets done faster, errors are reduced, and people can focus on more important, 
creative, or strategic tasks instead of routine chores.
