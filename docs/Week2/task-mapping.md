# Repetitive tasks with resepect to each department 

##  1. Sales / Booking Department

Tasks done repeatedly:

- Respond to customer calls or WhatsApp messages
- Check slot availability in the calendar
- Confirm booking if slot is free
- Request advance payment
- Record booking in schedule/calendar
- Send confirmation message to the customer
- Send reminders before match (e.g., 2 hours prior)
- Check on incomplete bookings

 

## 2. Finance  Department

Tasks done repeatedly:

- Record received payments
- Track unpaid bookings
- Issue refunds (according to cancellation policy)
- Update payment status in the system/calendar
- Generate daily revenue summary
  



## 3. Operations / Turf Management

Tasks done repeatedly:

- Turn on lights before matches
- Prepare turf for play (check nets, mark field)
- Monitor games to ensure rules and timing are followed
- Turn off lights after last match
- Clean turf after each match


## 4. Maintenance Department

Tasks done repeatedly:

- Inspect turf condition weekly
- Check lighting and other equipment
- Repair minor damages
- Schedule major maintenance work
- Ensure field is safe for players
